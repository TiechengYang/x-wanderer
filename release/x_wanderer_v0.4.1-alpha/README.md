# X Wanderer - 数字漫游者 Agent（结构化铁律核心版）

## v0.4.1-alpha 重新梳理后的状态

经过全面回顾与逻辑重构，移除了前期过度工程化的部分（理想画像自动生成、复杂偏好定制上下文等让代码显得臃肿的功能），专注于真正有效且果断的核心机制。

### 当前最核心的能力（ABC 激进方向）

1. **结构化铁律强制多轮回访战役**
   - Supervisor 在决策最开始检查 `active_revisit_targets`
   - 一旦有高价值目标且连续行动未过高 → **直接强制返回 engage**，完全绕过 LLM
   - 形成可靠的连续多轮对话，不容易半途逃脱

2. **analyze_people 强力驱动**
   - 全局分析后直接把高优先级人物全部灌入回访队列
   - 同时强行改写当前目标 + 大量注入具体子目标
   - 把目标状态拉到 needs_revision + 极低进度分，迫使系统重视关系维护

3. **干净的记忆闭环**
   - 每次 engage 后自动写入结构化关系总结
   - 关系图谱（节点 + 边类型 + 聚类 + 洞见）持续更新
   - Scheduler 可直接种子高价值回访对象

4. **测试友好**
   - `main_dryrun.py` 是主要测试入口，能清晰看到“分析 → 铁律强制多轮战役 → 目标被驱动”的全过程

---

**当前版本的目标是：让 Agent 在关系维护这件事上真正果断、可靠、不空转。**

跑干运行时重点观察铁律是否真的在强制连续 engage，以及目标是否被分析结果真实影响。

### 快速开始

```bash
# 测试模式（强烈推荐先看效果）
python3 main_dryrun.py

# 手动触发最强链路
echo "/analyze_people" > commands/human_input.txt
python3 main_dryrun.py
```

真实运行请先 `bash setup.sh` 并配置 `.env`。
